# Python Database Management System (Oracle 11g)

A Python-based database management application built with **cx_Oracle** for database connectivity and **Tkinter** for GUI.  
It supports **CRUD operations**, **predefined & custom queries**, **table creation**, and **real-time updates**.  
Designed to reduce manual database interaction time by ~40%.

---

## Features
- CRUD operations (Create, Read, Update, Delete)
- Table creation with dummy data population
- 10+ predefined SQL queries
- Custom query execution
- Range-based queries & joins across multiple tables
- Summary views for transactions and customer orders
- Real-time updates and deletion of records

---

- Oracle Instant Client (download from [Oracle](https://www.oracle.com/database/technologies/instant-client.html))
